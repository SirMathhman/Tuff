# Tuff — Simple TypeScript Starter

This is a minimal TypeScript project scaffold.

Prerequisites:

- bun (preferred) or Node.js + npm

Setup:

- Using bun:
  - Install dependencies: `bun install`
  - Run in dev: `bun run src/index.ts`
- Using npm:
  - Install dependencies: `npm install`
  - Dev: `npm run dev` (if you add a local dev runner)

Useful scripts:

- `build` — compile TypeScript to `dist`
- `start` — run compiled code from `dist`
- `dev` — run source with bun
- `lint` — run ESLint on `src`
- `format` — run Prettier
- `test` — run Jest tests

Happy hacking! ✨
