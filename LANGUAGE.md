# Language Spec — One Page ⚙️🔒

## 1) Purpose & Goals
- **Name (working):** Untitled systems language  
- **Primary goal:** Prevent insecure/unsafe code generated by agents for systems/embedded targets (prevent buffer overflows, use-after-free, data races, memory leaks).  
- **Priorities:** **Performance**, **Safety**, **Expressive types**, **Interop (C/LLVM)**.

---

## 2) Design Summary
- **Typing:** Static, **dependent/linear** types with ML-like inference.  
- **Memory:** **Zero-runtime**, **linear/move** semantics (no GC). Ownership enforced by the type system; no implicit runtime checks required for safety claims unless annotated.  
- **Verification:** **SMT-backed**, mostly-inferred proofs; `require`/`ensure`/`assert` annotations for explicit invariants.  
- **Concurrency:** Ownership-based across threads + linear types to statically prevent data races.  
- **Interop/Targets:** `extern` C-style FFI, LLVM backend, targets: bare-metal, Linux, Wasm, RTOS.  
- **Ergonomics:** Expression-oriented, **braced** blocks, `:` style type annotations, concise syntax, rich stdlib for systems.

> Note: Compiler prioritizes helpful diagnostics and fast incremental cycles; proofs are automatic where possible, with explicit hints when needed.

---

## 3) Core Syntax (examples)
- Bindings & functions:
  - `let x : I32 = 0;`
  - `fn add(a : I32, b : I32) : I32 => a + b;`
- Structs / methods:
  - `struct Point { x : I32; y : I32; }`
  - `impl Point { fn manhattan(&this) : I32 => abs(this.x) + abs(this.y); }`
- Externs:
  - `extern fn c_get() : I32;`
- Match / enums:
  - `enum Opt[T] { None, Some(T) }`
  - `match opt { Some(v) => v, None => 0 }`
- Verification annotations:
  - `fn safe_read(buf: &Buffer, idx: USize) : I8 requires idx < buf.len ensures true => ...`

---

## 4) Type System & Safety
- **Linear types / move semantics** for resources: values with linear kind must be consumed/moved; compiler enforces single-owner rules.  
- **Dependent fragments**: types can depend on values for sizes and bounds, e.g., `Buffer[n : USize]`.  
- **SMT integration:** solver drives discharge of assertions and refinement obligations; compiler *infers* many obligations and requests user hints (lemmas/`prove` blocks) when necessary.  
- **Safety properties enforced statically:** bounds, lifetime/moves (no use-after-free), race-freedom (no shared mutable aliasing), memory-leak freedom by default patterns + optional proof obligations for complex lifetimes.

---

## 5) Memory & Concurrency Model
- **No GC**, minimal runtime support for platform-required demarcation only.  
- **Ownership transfer** across threads via `move` semantics; shared immutables allowed (`&T` read-only borrows).  
- **Message-passing / channels** in stdlib built on top of ownership rules (safe by construction).

---

## 6) Verification Workflow
- **Mostly-inferred proofs**: compiler emits obligations; SMT tries to solve.  
- **User hints**: `lemma`, `prove { ... }`, and `#[hint]` annotations.  
- **Selective verification**: allow per-module `#[verify = full|partial|none]`.  
- **Runtime checks**: optional `--sanitize` mode to add checks when proofs are incomplete.

---

## 7) Tooling & Ergonomics
- Fast incremental compiler, **LSP** server, formatter, and REPL.  
- Errors: concise primary message + short actionable fix suggestions + `--explain` for extended diagnostics.  
- Small but useful standard library with safe buffer, device, and concurrency primitives.

---

## 8) Minimal EBNF (core)
```
program     ::= { item }
item        ::= fn_decl | struct_decl | enum_decl | extern_decl
fn_decl     ::= "fn" ident "(" [param_list] ")" [":" type] "=>" expr ";"
param_list  ::= param { "," param }
param       ::= ident ":" type
struct_decl ::= "struct" ident "{" { field } "}"
field       ::= ident ":" type ";"
enum_decl   ::= "enum" ident "{" { variant } "}"
extern_decl ::= "extern" "fn" ident "(" [param_list] ")" ":" type ";"
expr        ::= literal | ident | "match" expr "{" case_list "}" | block | call
```

- Safe buffer read (dependent size + inferred proof)
```rust
fn read_byte(b : Buffer[n : USize], idx : USize) : I8
  requires idx < n
  ensures true
=> b.data[idx]
```

